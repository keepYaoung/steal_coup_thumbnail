## Phase 1 — 수요 검증 (진행 중)

**목표:** 돈 쓰기 전에 유저가 진짜 원하는지 확인

### 실행 내용
- 현재 익스텐션 팝업에 AI 버튼 추가
  - `[AI] 이 상품 5각도 이미지 생성하기 (Beta)`
  - 클릭 시 토스트: "준비 중인 기능입니다. 출시 알림 받기 →"
- GA4 이벤트 추적으로 클릭률 측정

### 판단 기준
| 지표 | 기준 | 결과 |
|------|------|------|
| WAU 대비 클릭률 | 20% 이상 (약 27명+) | 측정 중 |
| 이메일 전환 | 10명 이상 | 측정 중 |

### 통과 시 → Phase 3 진행

## Phase 2 — 플랫폼 확장 (Phase 1과 병행)

**목표:** 글로벌 셀러 유입, 키워드 점유

### 실행 내용
- 알리익스프레스 / 테무 / 아마존 썸네일 추출기를 각각 별도 앱으로 출시
- 각각 다른 키워드로 웹스토어 노출 극대화
- 쿠팡 앱 하단에 크로스 프로모션 배너 추가
  - "알리/테무 버전도 있어요"

### 우선순위
1. **알리익스프레스** — 국내 셀러 접점 가장 높음
2. **테무**
3. **아마존**

### 기대 효과
- 기존 137명 → 신규 앱으로 자연 유입
- 앱별 독립 키워드 점유로 스토어 노출 극대화

## Phase 3 — 수익화 (검증 후 진행)

**전제 조건:** Phase 1 판단 기준 통과

### 옵션 A. AI 5각도 이미지 생성 (유료 기능) ⭐ 우선

- **구조:** 1회 결제, 5장 세트
- **가격대:** 2,900 ~ 3,900원 (심리적 저항 낮은 구간)
- **결제 방식:** 구글 웹스토어 인앱 결제 ← 정책 확인 필수
- **API:** 나노바나나 or 유사 서비스 — 실제 비용 직접 확인 후 마진 재계산 (AI 추정값 그대로 믿지 말 것)
- **레포 전환:** 결제 로직 추가 전 Private 전환 또는 서버 사이드 분리 필요

### 옵션 B. 애드센스 (보조 수익)

- 팝업 내 직접 삽입은 구글 정책상 까다로움
- 대안: 결과 화면을 새 탭(옵션 페이지)으로 열고 배너 삽입
- 현 규모에서 큰 수익 기대 어려움
- 브랜드 이미지 희석 리스크 있음 → 보조 수단으로만 고려
